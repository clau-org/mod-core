# mod-core

## Tasks

### Development

To run this module in development mode run:

```
deno task dev
```

### Test

To run the test for this module run:

```
deno task test
```

### Test in development mode

To run the test for this module in development mode run:

```
deno task test:dev
```

### Preview

To preview this module run:

```
deno task preview
```

## Release
